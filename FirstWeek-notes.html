Important Concepts

The basics of the web

The web is a collection of many computers. Within the web, computer can mutually send out and exchange data, text, pictures, music etc. A webpage is usually created in HTML, where people can get access to those different types of contents online. People ask computer to send out HTTP requests to URLs, which represent the location of computers (or server) online. The HTML content can be translated by web browser and displayed by the screen. HTML

HTML means Hypertext Markup Language. It is essentially the basic and major type of the online documents and materials. HTML consists of text and markup. They define what contents a webpage have and how the contents are shown to the users. We can refer to this page to learn more about HTML.

Important Tags

Tags play an key role in telling the web browser the type of the HTML element so that it knows how to correctly show a HTML webpage. Important tags include: b tag, em tag, a tag, (will start a new line)
br tag and so on.

Inline and Block

The HTML elements can be categorized as inline as well as block. For example, br tag is inline and p tag is block, which will usually provide an invisible box and put every piece of information inside. Showing an example picture
