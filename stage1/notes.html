The basics of the web

The web is a collection of many computers. Within the web, computer can mutually send out and exchange data, text, pictures, music etc. A webpage is usually created in HTML, where people can get access to those different types of contents online. People ask computer to send out HTTP requests to URLs, which represent the location of computers (or server) online. The HTML content can be translated by web browser and displayed by the screen. HTML

HTML means Hypertext Markup Language. It is essentially the basic and major type of the online documents and materials. HTML consists of text and markup. They define what contents a webpage have and how the contents are shown to the users. We can refer to this page to learn more about HTML.

Important Tags

Tags play an key role in telling the web browser the type of the HTML element so that it knows how to correctly show a HTML webpage. Important tags include: b tag, em tag, a tag, (will start a new line)
br tag and so on.

Inline and Block

The HTML elements can be categorized as inline as well as block. For example, br tag is inline and p tag is block, which will usually provide an invisible box and put every piece of information inside.

The Special Structure of HTML

The HTML documents have a special nested structure, like tree. Although this might sound intimidating in the beginning, it helps in organizing things into boxes at diffrent levels. In general, the tree-like structure comes from the idea of boxifying.

Several Ways to Include CSS

  • Write CSS in the <head> of HTML document
  • Link HTML to a seperate CSS file(in the same folder along with the HTML document)
  • Put down the styling code inline inside HTML

Summary of Playing with Boxes

Box Sizing

There are two ways to specify box sizes:

  • Set sizes in terms of percentage but not pixels, so that box sizes can be self-adpated when other changes are applied
  • Set the box-sizing attribute to border-box for every element.

Box Positioning

Divs are block elements, so they will take up the entire width of a page. Adding the rule display:flex; to the approriate CSS will override this behavior and let divs appear next to each other.

Useful tips

Revert Temporary Styling Changes

We can modify the style settings via Developer Tools. For example, we can change the margin of an invisible box. If we don't like the changes, refreshing the page should be able to help us revert the changes.
